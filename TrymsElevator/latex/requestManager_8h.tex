{\footnotesize\ttfamily void get\+New\+Request (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Provides the next request to the elevator based on logic. The request is represented by the floor given as an int. 

It first checks for requests on the elevators current floor.

When there is no request on its current floor it loops through the rest of the database. If the elevator is on floor 1 or 2, it loops through the database from floor 4 and downwards. If the elevator is on floor 3 or 4, it loops through the database from floor 1 and upwards.

The new request is added directly to the elevators current request.
